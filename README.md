# code-edit-api

I want an app where I can put in some code and a prompt for how to edit it and then it calls an api and shows the new code with diff

Call localhost:8080/edit with a json of the code + prompt

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/code-edit-api.git
cd code-edit-api
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
